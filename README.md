# Amazon Outposts (amazon-outposts)
AWS Outposts is a family of fully managed solutions delivering AWS infrastructure and services to virtually any on-premises or edge location for a truly consistent hybrid experience. It allows you to extend AWS infrastructure, AWS services, APIs, and tools to virtually any data center, co-location space, or on-premises facility.

## APIs

### AWS Outposts API
The AWS Outposts API provides programmatic access to create and manage Outposts, sites, orders, catalog items, assets, and local gateway routes for deploying AWS infrastructure on-premises.
